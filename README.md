<h1>🐾 PetStar</h1>

Plataforma web para adoção de cães e gatos de ONGs, desenvolvida como Trabalho de Conclusão de Curso.

**📌 Sobre o projeto**

O PetStar foi criado com o objetivo de conectar ONGs a pessoas interessadas em adotar animais, facilitando o processo de adoção de forma digital, organizada e acessível.
A plataforma permite visualizar animais disponíveis, acessar detalhes e iniciar o processo de adoção.

**Funcionalidades:**

📋 Listagem de animais disponíveis</br>
🔍 Filtros de busca (tipo, porte, etc.)</br>
🐶 Página de detalhes do animal</br>
🏢 Cadastro de ONGs e Adotantes</br>
📩 Envio de solicitações de adoção</br>
📧 Notificações por e-mail</br>
🔐 Sistema de login para ONGs e Adotantes

**🛠️ Tecnologias utilizadas:**

Frontend: HTML, CSS, JavaScript</br>
Backend: PHP</br>
Banco de dados: MySQL</br>
Outros:</br>
PHPMailer (envio de e-mails)</br>
Typeform (coleta de dados)
